VisionHome split v1

Files:
- index.html  -> struttura pagina
- styles.css  -> tutti gli stili
- app.js      -> tutta la logica JavaScript

Come usarli:
1. Tieni i 3 file nella stessa cartella
2. Apri index.html oppure caricali sul tuo server
3. Se in futuro vuoi, il passo successivo è separare app.js in:
   - auth.js
   - calendar.js
   - dealers.js
